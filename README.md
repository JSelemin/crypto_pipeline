# Crypto Data Pipeline

Pipeline de datos que extrae información sobre cinco criptomonedas de la API de CoinGecko, las guarda como archivos Parquet y utiliza SQL (DuckDB) para transformar los datos.

### Tecnologías

- Python

- pandas

- requests

- Parquet (PyArrow)

- DuckDB

- SQL
